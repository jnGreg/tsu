# txt_sum_up_proj

##  Project Description
A simple REST API app to summarize texts given in the form of a URL or pasted.

## Technologies
- [Python] - Web programming language, version 3.9
- [Flask] - Web App Framework
- [Heroku] - Hosting platform
- [Docker] - Docker
- [REST API] - REST API
